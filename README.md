# Entrega-3
Tercera entrega para el curso  "Computacion para Econocmistas" de la Universidad de Costa Rica

Este programa fue hecho por un estudiante de la Universidad de Costa Rica
durante el II ciclo 2016, para el curso Computación para Economistas.
El programa se encuentra en la web con la intención que cualquiera lo
pueda utilizar.

A continuación se explican los pasos seguidos durante la realizacion
del programa:

I. Se solicita al usuario que ingrese un numero del 1 al 6, solo esos
numeros seran validos.

1. Factorial
Se utiliza una sentencia while para poder realizar un factorial.

2. Producto de Wallis
Se utilizo la sentencia While para poder calcular el producto de Wallis
hasta el numero solicitado por el usuario.

3. Diagonal
Con la sentencia While se creo una diagonal de asteriscos ("*"),
esta es la diagonal principal.

II. Se va a trabajar ahora con numeros primos
Se solicta al usuario que vuelva a ingresar un numero cualquiera.

1. Mediante la formúla (2^p) - 2| p, primo se evaluara si el numero
ingresado es primo.

2. Ya que no existe una formula para generar numero primos, el programa considera
numeros primos aquellos que no sean divisibles por 2, 3, 5, 7 y 11.
